¿Qué es un enlace en un sitio web?

Una de las cosas que caracteriza a la web como el medio de comunicación es la navegación por medio de hipervínculos (links o enlaces). Por lo tanto esta conexión entre archivos permite a cada usuario elegir el sentido de la navegación para así poder tener una conexión. En HTML estos enlaces se marcan con la siguientes etiqueta y el atributo principal es href="" en ella se encuentra la ubicación del archivo que se encuentra en la misma carpeta que el archivo que lo está llamando. 

En las siguientes etiquetas <a href=""></a>podemos colocar cualquier
elemento html que funcionará como botón, generalmente se coloca un texto 
o una imagen.
