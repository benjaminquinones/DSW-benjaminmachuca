    Descripcion del sitio

    El proposito de este sitio web es dar a conocer las estructuras de codigo, listas y que son enlaces. Además de eso también creé esta pagina para que las personas puedan ver lo que se puede hacer usando HTML Y CSS Basico.

    HTML Y CSS sirven más que nada para darle una estructura y diseño a nuestra pagina, ya que una pagina web contiene FRONT-END Y BACK-END El front-end viene siendo la parte de estructuras como HTML pero el Back-end es la parte de codificacion y validicacion de datos o peticiones.

    Recomendaciones para tu pagina web


    La primera recomendación que yo te daría al empezar a crear tu pagina es: 

  •  Utilizar el  <meta  name ="description" content="Aquí va la descripcion de tu pagina web para que sea visible en google">
  •  Utilizar el  <meta  name ="author" content="Aquí pondrás tu nombre para que aparezcas como el creador del sitio web">
  •  Utilizar el  <img src="tu imagen"  alt="aquí se pone la descripcion de la imagen para que aparezca más facil en google" >
